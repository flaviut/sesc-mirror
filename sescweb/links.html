Main Users

  • IACOMA : SESC was build in the IACOMA group to substitute previous simulation tools.
  • FlexRAM The FlexRAM group is a project of the IACOMA group specialized in processors and memory. SESC is the new current simulation environment for FlexRAM.
  • M3T The M3T group another project of the IACOMA group. It has as objective to create a chip multiprocessor with extra hardware so that it can execute non-parallel applications.

Related Projects

  • simplescalar Simplescalar is the main simulator in the computer architecture.
  • RSIM The only simulator, as far as I know, that really executes the instructions out-of-order. RSIM is another simulator from UIUC.
