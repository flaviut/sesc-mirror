CVS Server from sourceforge

In the CVS server there are two main packages the web server itself (sescweb), and the simulator (sesc). It is possible to browse the source code
Anonymous CVS Access
This project's SourceForge CVS repository can be checked out through anonymous (pserver) CVS with the following instruction set. When prompted for a password for anonymous, simply press the Enter key.

cvs -d:pserver:anonymous@cvs.sesc.sourceforge.net:/cvsroot/sesc login

cvs -z3 -d:pserver:anonymous@cvs.sesc.sourceforge.net:/cvsroot/sesc co sesc

Updates from within the module's directory do not need the -d parameter.
